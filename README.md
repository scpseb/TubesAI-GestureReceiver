# TubesAI-GestureReceiver
Untuk keperluan Tugas Besar mata kuliah Artificial Intelligence, Sistem Internet of Things, dan Sistem Tertanam

Receiver sistem berfungsi sebagai unit aktuator yang menerima sinyal kontrol dari XIAO ESP32-S3 menggunakan BLE untuk menyalakan atau mematikan lampu DC 12V dan kipas DC 12V menggunakan relay 2-channel dan menampilkan kondisi lampu dan kipas pada web dashboard menggunakan Blynk
